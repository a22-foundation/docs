# A22 Documentation Site

The official documentation for A22, built with [Docusaurus](https://docusaurus.io/).

## Content
- **Introduction**: Overview of A22 philosophy (from Whitepaper).
- **Specification**: Complete language reference.
- **Getting Started**: Guides for installing and using the runtime.

## Development
```bash
npm install
npm start
```
This command starts a local development server on port 3000.

## Build
```bash
npm run build
```
